Приложение сделано на Flask. Частично ручками, частично навайбкожено

## Реализовано

- Модуль 1: база данных в 3НФ, PK/FK, ссылочная целостность, etc
- Скрипт схемы БД: `docs/database_schema.sql`.
- Импорт Excel в SQLite: `scripts/init_db.py`.
- Модуль 2: страница входа, роли (`guest/client/manager/admin`), просмотр товаров.
- Модуль 3:
  - поиск, фильтр по поставщику, сортировка по остатку;
  - добавление/редактирование/удаление товаров (admin);
  - запрет удаления товара, если он есть в заказе;
  - переходы по страницам.
- Заказы (для manager/admin): страница просмотра.
- Подключен линтер `flake8`.

## Для запуска

```bash
python3 -m pip install -r requirements.txt
python3 scripts/init_db.py
python3 src/webapp.py
```

По-умолчанию подрубает по: `http://127.0.0.1:5000/login`

## Линтер

```bash
python3 -m flake8 src scripts
```
